# consent-form

A static HTML mockup of a data-consent artifact form, modeled on the consent schema used in Data Empowerment/Consent frameworks (fields for "Data Principal," "Artifact Expiry," and purpose codes like B1–B6, P1–P5, G1–G5 for agricultural-data use cases such as credit, insurance, logistics, and farm advisories).

**Scope:** UI markup only — no JavaScript, no form submission, no backend or cryptography in this repo.

## Tech Stack

- HTML only

## Structure

```
index.html   - static consent form markup (AIP / AIU / Data Principal / purposes / item details / expiry)
```
